# 5631AS — 3-digit 7-segment display 0.56" | KiCad STEP model

3D модель для KiCad — 3-разрядный 7-сегментный индикатор 5631AS, 0.56 дюйма.
Сделана вручную в FreeCAD 1.1.1.

## Файлы
- `5631AS_3digit_7seg_0.56in.step` — STEP модель для KiCad
- `Indikator.FCStd` — исходник FreeCAD (можно редактировать)

## Характеристики
- Высота цифры: 14.2 мм (0.56")
- Корпус: 37.8 × 19.0 × 8.0 мм
- Шаг контактов: 2.54 мм
- Количество контактов: 11
